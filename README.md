# Fruit Classification Project

## Project Overview
This project performs data preprocessing and supervised machine learning for fruit classification.
The dataset includes 6 raw CSV datasets (apple, banana, grape, and 3 mixed fruit datasets).

## Folder Structure
- data/raw/ -> original datasets
- data/intermediate/ -> combined and cleaned datasets
- data/final/ -> final preprocessed dataset + train/val/test splits
- src/ -> preprocessing and ML scripts
- models/ -> saved trained models
- reports/ -> report document and figures
- outputs/ -> results and evaluation metrics

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py
